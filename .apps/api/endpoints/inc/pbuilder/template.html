Akai FORCE/MPC Chord Progression Builder

Progressions Library

Help & Instructions

  • Purpose: Solely To Generate Akai Force/Mpc Note Mode (Progreessions) Compatible Progresison Files.
  • This is not a Chord/Progression Generator/Midi Converter. (Tools already Exist for those!)
  • Note: This is Personal Tool that I built and am sharing with community.
  • Click the Scale/Chord Degrees in Left Panel To Build the Chord Progressions
  • It is Up to You to Select Correct Chord Degrees for the Scale/Song
  • After your are done building/previewing the progressions, Click Save to Save as Akai format Progression file.
  • To add Category to Progresisons list just create the name as "CategoryName-ProgressionName"
  • Use MUTE to Enter Separators if you Want to Subdivide the Progression into verse cgorus section. Mute will trigger from the Note: 0 (Lowest Note)
  • The +/- Button allow you to shift the octave of the chord relative to the Progression octave.
  • You want to check out Hooktheory.com Song Database to find some progressions to try
  • Midi Output is Omni and is sent to all 16 Midi Channels!
  • Some Bugs are expected and will be fixed as I find them.
  • More Features Refinements will be added as I find the need
  • ~Enjoy!!
